# OpenGL

## Install OpenGL

`sudo apt install mesa-utils libglu1-mesa-dev freeglut3-dev mesa-common-dev libsdl2-dev libdbus-1-dev`

## Check OpenGL Version

`glxinfo | grep "OpenGL version"`

## Test OpenGL

`vblank_mode=0/1 glxgears` disable/enable Vsync
